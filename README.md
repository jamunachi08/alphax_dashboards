# AlphaX Dashboards

Installable Frappe/ERPNext app providing management web dashboards (CRM, Finance, HRMS).

## Install

```bash
bench get-app <repo_url>
bench --site <site> install-app alphax_dashboards
```

## Pages
- `/alphax-crm-dashboard`
- `/alphax-finance-dashboard`
- `/alphax-hrms-management-dashboard`
- `/alphax-hrms-self-service-dashboard`

All pages require login.
